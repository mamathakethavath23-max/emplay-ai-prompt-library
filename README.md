# AI Prompt Library (Submission Ready)

A full-stack web application to organize and manage AI image generation prompts using Angular, Django, PostgreSQL, Redis, and Docker Compose.

---

## 🚀 Project Overview

This application allows users to:

* View saved prompts
* Add new prompts
* Open prompt details
* Track prompt views using Redis
* Manage prompts in a structured dashboard

It solves the problem of storing prompts in random notes or chats by keeping everything in one organized system.
